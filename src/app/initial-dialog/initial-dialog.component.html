Bem-vindo ao Campo Minado! 💣

O objetivo do jogo é descobrir onde estão todas minas.

1. Começando

Clique em qualquer campo para abri-lo. Se for uma mina, o jogo termina. 💥

2. Bombas ao Redor

O número em cada campo aberto revela quantas minas estão ao seu redor. Assim você pode tentar adivinhar quais campos são seguros para clicar. 🤔

3. Marcando Minas

Para marcar um campo que você acredita conter uma mina, clique com o botão direito do mouse. Isso colocará uma bandeira no campo, ajudando você a lembrar que ali pode haver uma mina. 🚩

4. Ganhar o Jogo

Você vence quando todos os campos que não contêm minas estiverem abertos! 🏅

Que a sorte esteja sempre a seu favor!
